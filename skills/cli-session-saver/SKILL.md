# CLI 会话落盘规范

## 触发条件
每次回复前自动执行，无需用户提醒。

## 核心规则

### 1. 会话存档
- 路径：`~/.hermes/cli-sessions/YYYY-MM-DD.md`
- 格式：每轮对话用 `### 对话轮 N — 时间戳` 标记
- 内容：包含用户输入 + AI输出

### 2. 触发时机
- 每次回复的**最后一个工具调用**必须是 `write_to_file` 落盘
- 存档写成功后才发消息给用户

### 3. 验证机制
- 写完后用 `execute_command` + `ls -la` 验证文件存在
- 如果验证失败，立即告知用户

## 存档模板
```
### 对话轮 N — YYYY-MM-DD HH:MM

**用户**：<用户输入>

**AI**：<关键回复摘要/产出物>

---
```

## 注意事项
- 不要声称"已完成"除非验证通过
- 验证失败要说"出错了，需要重试"
